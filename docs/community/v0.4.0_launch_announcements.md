## 🇨🇳 中文 — long form (微信公众号 / 知乎专栏 / 实验室内部邮件)

```
# pyALDIC v0.4.0 发布 —— 多语言 DIC 软件向全球用户开放

pyALDIC 是一款基于 Augmented Lagrangian（增广拉格朗日）方法的
Python 数字图像相关（DIC）软件，支持自适应四叉树网格加密，
配套完整的桌面 GUI。v0.4.0 的核心更新是让 pyALDIC 真正走向
全球 DIC 社区。

## 本次更新亮点

### 🌍 8 种语言 100% 完整翻译

GUI 界面已完整支持：
- 简体中文 / 繁體中文
- 日本語 / 한국어
- English / Deutsch / Français / Español

切换方式：顶部菜单 → 设置 → 语言。重启后全界面生效。

术语翻译经过领域校对（子集、位移、应变、网格、加密、种子点、
参考帧等词汇均对应中文力学教材标准）。

### 🎥 端到端 78 秒演示视频

README 内嵌，完整走通：
  导入图像 → 选择工作流 → ROI 绘制 → 网格加密
  → 运行 DIC → 位移场 → 应变场

其中 "选择工作流" 和 "运行 DIC" 关键步骤有 2× 放大 + 磨砂
背景聚焦特效，方便用户看清操作细节。

### 📚 README 大改版

对比表格（与 Ncorr / DICe / VIC-2D / MatchID）去除营销夸大，
新增技术深度行（公式表述 / 网格 / 3D 立体 DIC 支持）。
Accuracy 章节不再自报 RMSE，改为指向同行评议文献：

- Yang & Bhattacharya (2019) "Augmented Lagrangian Digital
  Image Correlation" Experimental Mechanics
- Tong et al. (2025) "3D Stereo Adaptive Mesh Augmented
  Lagrangian Digital Image Correlation" Experimental Mechanics
- DIC Challenge 2.0（Reu et al.）社区基准

### 🔤 CJK 字体显著改善

旧版在 Windows 下中文界面字体默认回退到宋体系列，显示发虚。
v0.4.0 加入了完整字体回退链（微软雅黑 / 苹方 / Noto Sans CJK
/ Yu Gothic / Malgun），中日韩三语界面和 matplotlib 绘图
标签都会使用现代无衬线字体。
```
